Projects

Food Order Website

Developed a full-stack online food ordering system that allows users to browse the menu, add items to the cart, and submit orders by filling out personal information. Admins can review, confirm, or reject orders via an admin panel, with detailed views of customer selections and pricing. The frontend was built using React.js, TypeScript, and the backend was built using Node.js, and integrates with Mailtrap for receiving customer order notifications via email.and integrated Firebase for data storage.

YelpCamp

Developed a full-stack website as part of an online course enabling users to register, log in, log out, upload campground locations and images, and post comments. Utilized Express and Node.js for the back end, and integrated a database for data storage.

Gomoku

Developed a fully playable Gomoku game in Python using Processing. Implemented key gameplay mechanics, including alternating turns, win detection, and AI opponent logic. Applied object-oriented principles and score tracking, and utilized unit testing (Pytest) to ensure reliable game functionality.

PacMan

Developed a fully playable Pac-Man game in Python using Processing. Implemented key gameplay mechanics, including character animation, enemy AI, and collision detection. Applied object-oriented principles and unit testing (Pytest) to ensure robust gameplay and functionality.
